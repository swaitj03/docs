<Card
title="Our Idea"
icon="lightbulb"
horizontal>
    WitsEats is a campus-based food delivery web app that connects students at the University of the Witwatersrand with on-campus restaurants. The platform allows students to order meals directly to their building and floor, using real-time geolocation and live chat for precise delivery instructions.
</Card>

<Card title="Features"
icon="gears"
horizontal>
Interactive map for selecting delivery location on campus <br/>
Geolocation-based order restriction (only allow orders from within campus) <br/>
Restaurant browsing with searchable menus <br/>
Real-time chat between customer and delivery driver <br/>
Live order tracking with status updates <br/>
Role-based interfaces for customers, drivers <br/>
Admin dashboard for managing orders, delivery zones, review and manage drivers <br/>
Signup for drivers <br/>
Secure authentication and payments 
</Card>

<Card title="APIs"
icon = "cloud"
horizontal>
Geolocation API <br/>
Map Rendering API <br/>
Authentication API <br/>
Realtime Messaging API <br/>
Payment API
</Card>
