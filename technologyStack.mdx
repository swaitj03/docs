Frontend: React (hosted on vercel) <br/>
Styling: shadcn/ui <br/>
Backend: Supabase + FastAPI(hosted on render) <br/>
Authentication: Supabase Auth <br/>
Database: PostgreSQL (via Supabase) <br/>
CI/CD: GitHub Actions <br/>
Testing: Jest(frontend) and Pytest(backend) <br/>
Package managers: npm(frontend) and uv(backend) <br/>
