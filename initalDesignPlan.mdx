<Card
title="Our Idea"
icon="lightbulb"
horizontal>
    WitsEats is a campus-based food delivery web app that connects students at the University of the Witwatersrand with on-campus restaurants. The platform allows students to order meals directly to their building and floor, using real-time geolocation and live chat for precise delivery instructions.
</Card>

<Card title="Features"
icon="gears"
horizontal>
Interactive map for selecting delivery location on campus

Geolocation-based order restriction (only allow orders from within campus)

Restaurant browsing with searchable menus 

Real-time chat between customer and delivery driver

Live order tracking with status updates

Role-based interfaces for customers, drivers

Admin dashboard for managing orders, delivery zones, review and manage drivers

Signup for drivers
Secure authentication and payments
</Card>

<Card title="APIs"
icon = "cloud"
horizontal>
Geolocation API

Map Rendering API 

Authentication API 

Realtime Messaging API 

Payment API
</Card>

<Card title="UI Ccomponents"

Chat interface for communicating with delivery driver

Order status and tracking page

Delivery instructions input form

Vendor dashboard for managing incoming orders

Admin dashboard with analytics and settings
